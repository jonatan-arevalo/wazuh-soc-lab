# Deployment Process

## Wazuh Server

1. Installed Ubuntu Server 24.04
2. Configured network connectivity
3. Installed Wazuh using the official installer
4. Verified dashboard accessibility

## Active Directory

1. Installed Windows Server 2022
2. Promoted server to Domain Controller
3. Created domain lab.local
4. Configured DNS
